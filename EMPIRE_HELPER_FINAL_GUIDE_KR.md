# 🤖 디지털엠파이어 II 스마트 도우미 '엠파이어 도우미' 최종 운영 가이드

이 문서는 **2026년 주차 요금 개정안**과 **최신 파스텔 테마 UI**, 그리고 **GitHub Actions 자동 배포 시스템**을 포함한 전체 운영 매뉴얼입니다.

---

## 🚀 1. 핵심 서비스 개요

### 🚗 2026년 주차 요금 로직 (개정)
- **일일권 (12시간)**: 10,000원 (1회 최대 3장 사용 가능)
- **시간권 (1시간)**: 1,000원 (1회 최대 2장 사용 가능)
- **중복 사용 (Stacking)**: 일일권과 시간권을 섞어서 사용 가능 (예: 26시간 주차 시 일일권 2장 + 시간권 2장 적용)
- **추가 요금**: 할인권 적용 후 남은 시간은 **1시간당 1,500원**의 일반 요금 부과.
- **UI**: 친근한 파스텔 블루 및 소프트 옐로우 톤 적용.

### 🗨️ 지능형 챗봇 (Antigravity 에이전트)
- **프롬프트 기준**: 2025 관리규약 및 2026 주차 로직 동기화.
- **클릭 활성화**: 우측 하단 챗봇 버튼 및 퀵 카드 클릭 시 개발 중 안내 메시지 출력.

---

## 🛠️ 2. 개발 및 유지보수

### 📂 주요 파일 구조
- `app/page.tsx`: 메인 대시보드 및 상호작용 로직.
- `components/ParkingCalculator.tsx`: 개정된 주차 로직 엔진.
- `lib/parking-logic.ts`: 요금 계산 핵심 알고리즘.
- `.github/workflows/ci.yml`: GitHub Actions 자동 배포 설정.

### 🔄 배포 프로세스 (GitHub Actions)
현재 배포는 **GitHub Actions**를 통해 자동으로 수행됩니다.
1. `master` 브랜치에 코드 푸시.
2. Actions가 자동으로 `npm run build` 실행.
3. 빌드 결과물(`out` 폴더)이 GitHub Pages로 자동 배포.
> **주의**: GitHub Settings -> Pages에서 Source를 `GitHub Actions`로 설정해야 합니다.
